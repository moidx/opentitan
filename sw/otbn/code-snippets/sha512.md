# SHA2-512

The SHA-512 implementation is compliant with the NIST FIPS 180-4 specification.


## Internal block processing size

The internal message block processing size is 1024 bits. The function assumes
that all calls have the correct block size and expected padding.

## Implementation notes

Steps:

1.   Set the initial hash value. See constants (section 5.3.5). this is to be
     handled in the sha_512_init function.
2.   The message is padded and parsed as specified in section 5. This is to be
     handled in the sha_512_final function.
3.   For each block:
     1.   Prepare the message schedule.
     2.   Initialize the eight working variables a, b, c, d, e, f, g, and h with
          the hash i-1 value.
     3.   Calculate T1, T2, h, g, f, e, d, c, b, a
     4.   Calculate the i-th intermediate hash value.
